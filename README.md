## Build Project
first install docker depend on your system  
docker-compose up -d --build  
-d to work in background  

## install project
git clone https://github.com/mohamedjs/digsiy.git  
docker-compose exec php bash 
composer install  
npm i 
npm run dev  
php artisan migrate --seed
